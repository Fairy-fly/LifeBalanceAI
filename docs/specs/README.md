# LifeBalanceAI Project Specifications

This directory contains the current project specification documents used for
requirements, architecture, implementation notes, and API integration reference.

## Current Documents

- `LifeBalance_AI_Requirement_Specification_V3.0.docx`
  - Requirement specification for the current LifeBalanceAI scope.
  - Describes product goals, user roles, functional requirements, boundaries,
    and the AI-driven value model.
- `LifeBalanceAI_系统总体设计.docx`
  - System design document.
  - Describes architecture, module boundaries, database design, and page flow.
- `LifeBalance_AI_Dev_Manual_V3.0.docx`
  - Development manual for the current implementation direction.
  - Summarizes technical choices, development roadmap, and module behavior.
- `LifeBalanceAI_API接口调用知识点总结.docx`
  - API knowledge summary.
  - Explains how the project calls AI APIs, packages requests, parses responses,
    handles errors, and stores generated results.

## Archived Versions

Older V2 documents are kept in `docs/archive/specs/` for historical reference:

- `LifeBalance_AI_Dev_Manual_V2.0.docx`
- `LifeBalance_AI_Requirement_Specification_V2.0.docx`
