# Bundled Font Sources

The app bundles these fonts for consistent UI and exported report rendering:

- MiSans Regular, Medium, Bold: downloaded from Xiaomi HyperOS font download (`https://hyperos.mi.com/font-download/MiSans.zip`).
- Smiley Sans Oblique: downloaded from the `atelier-anchor/smiley-sans` GitHub release `v2.0.1` (`smiley-sans-v2.0.1.zip`).

Review each upstream license before commercial redistribution. The files are included here so the desktop app, generated AI text, and exported report images do not depend on system-installed fonts.
